# SEO Checklist (à copier-coller avant chaque publish)

```
NIVEAU 1 (bloquant)
[ ] <title> ≤ 60 caractères, mot-clé en tête
[ ] <meta name="description"> 140-160 caractères
[ ] <link rel="canonical">
[ ] <h1> unique
[ ] OG image présente (1200×630+)
[ ] hreflang pour langues disponibles + x-default
[ ] Au moins 1 JSON-LD valide

NIVEAU 2 (warn)
[ ] 3+ liens internes
[ ] 1+ lien externe vers source autorisée
[ ] <meta name="keywords"> 8-15 mots
[ ] Twitter Card complète
[ ] Images alt text + lazy load (sauf hero)

NIVEAU 3 (nice)
[ ] Breadcrumb JSON-LD
[ ] FAQ JSON-LD si pertinent
[ ] Article ≥ 600 mots
[ ] IndexNow ping après publish
```
